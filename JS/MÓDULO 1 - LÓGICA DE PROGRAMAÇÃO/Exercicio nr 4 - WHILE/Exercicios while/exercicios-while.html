1. Escreva um programa que solicita ao usuário que adivinhe um número entre 1 e 100. O programa deve informar se o número digitado é maior ou menor que o número a ser adivinhado. O programa deve continuar pedindo ao usuário para adivinhar até que ele acerte o número.

2. Escreva um programa que solicita ao usuário que digite uma senha. O programa deve continuar solicitando uma senha até que o usuário digite a senha correta.

3. Escreva um programa que solicita ao usuário que digite um número. O programa deve calcular a soma de todos os números de 1 até o número digitado pelo usuário e exibir o resultado.

4. Escreva um programa que solicita ao usuário que digite uma palavra. O programa deve contar e exibir o número de vogais na palavra digitada.

5. Escreva um programa que solicita ao usuário que digite um número. O programa deve calcular e exibir o fatorial do número digitado pelo usuário.
